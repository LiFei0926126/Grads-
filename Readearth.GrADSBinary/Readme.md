使用须知
1.该类库尚在开发中，如使用遇到bug，请记录下来，并反馈。
2.《TDEF》在解析时由于各模式数据处理人员的习惯不同会有不同的格式，暂时兼容2种模式。
如无必要可在ctl文件内移除。
************************
hh:mmZddMMMyyyy        16:00Z12MAY2015       2015年5月12日 16时00分
hhZddMMMyyyy                16Z12MAY2015            2015年5月12日 16时
************************
3.ctl文件中《PDEF》为必要项。
4.该类库属于x86架构，再添加引用时需要注意。
